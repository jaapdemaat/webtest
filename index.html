What you need to get started

1. Web browser

A web browser, or simply "browser," is an application used to access and view websites. Common web browsers include Microsoft Internet Explorer, Google Chrome, Mozilla Firefox, and Apple Safari.
If you haven't already got a web browser on your computer please use the follow links to download one. We recommend the Google Chrome browser but you are free to use any including ones not listed below.

2. text editor

A text editor is a type of computer program that edits plain text. Microsoft Windows systems come with the simple Notepad, Apple's Mac OS come with TextEdit. While both are perfectly fine for writing HTML & CSS, many people-especially programmers-prefer other editors with more features. We recommend using the Sublime editor but you are free to use any editor, including ones not listed below.
